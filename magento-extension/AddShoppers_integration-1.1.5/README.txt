[Addshoppers integration for Magento]

NOTE: Please backup your magento installation before installing, we are not responsible for errors during manual installation.

Installation instructions. 
---------------------------

1. Copy the "app/" directory into the root Magento folder.

2. Copy the "skin/" directory into the root Magento folder.

3. Clear your magento cache.

4. Log out then log back into admin.

5. Go to system -> configuration >> Catalog >> [Addshoppers] enable the extension, click save, then login with your addshoppers credentials.

If you have any issues or questions please email The Big Cat at [help @ addshoppers.com]
